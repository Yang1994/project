What is HDT(HuDong Tap)?

The HDT is a decentralized interactive traffic sharing service.

The HuDong Tap is an incentive application community based on the decentration of the block chain. Application developers publish a paid reward task based on interactive behavior to catch users' attention. After the users gets the task, they will experience the product. According to the intelligent contract, the workload of the user's valid interaction is proved, and the Token award is issued.Interactive coin will integrate application developers and users in a decentralized community to create an autonomous ecosystem, and to fully inspire the value creation of users and application developers.

Innovation

HDT(HUDONG TAP）

HDT dedicated to let eco-users hold token tokens based on blockchain and smart contracts through decentralized design. Using their own interactive behaviors to vote and spend on apps，To create an open and fair democratic ecosystem.

  • Under the decentralized account system

    the decentration of account system designed by Ethereum that allows user accounts to be no longer controlled by any organization to make users can freely select any application and link all the applications through the decentration account.

  • Interaction produces value

    By validly motivating users' interactive behavior, highly dynamic interaction acts as the basis for incentive selection, it fully stimulates the value creation of users and application developers.

  • Innovative evaluation system

    User interaction in application is an valid proof of activity degree. Therefore, by collecting valid user interaction behavior, it can provide users with reference as activity evaluation. Activity will be an important part of evaluation system.

  • Intelligent recommendation system

    innovatively combines the three basic algorithms above and the interaction behavior based on the block chain. These interactive behaviors are important data for personalized intelligent recommendation systems.
